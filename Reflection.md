In CSS position absolute makes it any text or image placed has a fixed position that we can choose. If it was removed the entire webpage will break. For example, in the magazine webpage I had to create for this assignment, when i removed position absolute in magazine-cover p, the entire text on the cover page completely broke. All the text have defaulted to the upper left corner. In CSS ::before just to insert an object before the actual selected text or element. For example, in the webpage the selected element has text, in this case the headings. And ::before was used to generate black boxes that would be placed behind the text. The biggest challenge I faced when positioning the elements on the cover was trying to figure out why my KENDRICK LAMAR heading and paragraph about contributors were all squished and on top of each other. Eventually, I solved the problem by decreasing magazine-cover h2 left: to 493px. And I added a line height of 1.4 to magazine-cover p.
