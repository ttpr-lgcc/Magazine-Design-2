# Reflection

## 1.

Position absolute removes an element from the normal page flow and lets me place it exactly where I want using top, left, right, and bottom. It is useful for the magazine cover because text needs to sit in precise locations like a real printed magazine. Without it, the text would stack normally and ruin the layout.

## 2.

The ::before pseudo-element adds content before an element without adding extra HTML. It is useful for decorative bars behind the text because it keeps the HTML cleaner and separates design from structure.

## 3.

The hardest part was lining up the text exactly like the sample image. Small pixel changes made a big difference. I used Chrome DevTools to inspect elements and test top/left values live until everything looked correct.
